Mi az RKS, miről szól ez az oldal?

2x2-es Rubik kocka kirakó program. A programot Java programozási nyelvet tanuló illetve az objektum orientált programozással ismerkedő diák első projektje. Körülbelül 2-3 napi munka óra alatt készült el a program. A legnehezebb rész a megoldó algoritmus volt.
A program jelenleg letölthető NetBeans projekt formájában és ott futtatható parancssoros környezetben.

Letöltés

A program működése:

A program 6db 2x2-es tömböt használ mint a kocka oldalai. A színek karakter formában vannak tárolva pl.: Kék - k
12 forgatási metódust tartalmaz a program. Úgy kell elképzelni mintha egyik oldalt magunk felé tartanánk és ahhoz viszonyítva jobb oldalt felfele vagy lefele tudjuk forgatni. forgatási metódus pl.: forgJf(); vagy forgJl();
Ezek után egy keverési metódus a létező forgató metódusok segítségével véletlenszerűen összekeveri a kockát
A program legnagyobb része a kockát megoldó metódus mely az algoritmus fülön található meg kifejtve.
